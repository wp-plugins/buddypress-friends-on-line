== Description ==

Plugin will display on your Friends page a new tab called Online with a list of currently online friends.

If will also create a widget that will do the same.

Visit [CosyDale.com](http://cosydale.com/my-plugin-buddypress-friends-on-line.html) for more information.

Please rate this plugin. I will develop it further, but need some feedback.

== Installation ==

1. Upload plugin folder `/bp-friends-on-line/` to the `/wp-content/plugins/` directory
1. Activate 'BuddyPress Friends On-line' through the 'Plugins' menu in WordPress
1. That's all.
